## 🔬 ADX策略说明

### 核心逻辑
```python
# 计算市场ADX(14)
adx = calculate_adx(market_prices, period=14)

if adx > 25:
    use TrendFollowingStrategy()  # 强趋势 → 吃主升浪
elif adx < 20:
    use MeanReversionStrategy()  # 震荡 → 博反弹 (或空仓)
else:
    use MultiFactorStrategy()    # 中性 → 多因子
```

### 两版本对比

| 策略 | ADX>25 | ADX<20 | 适用场景 |
|------|--------|--------|----------|
| 自适应ADX | 趋势跟踪 | 均值回归 | 全天候交易 |
| 自适应ADX+空仓 | 趋势跟踪 | **空仓** | 风险厌恶 |
