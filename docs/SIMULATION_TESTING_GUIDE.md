## 🚀 自动化方案

### 方案A: 自动化定时任务（推荐）

#### 1. 设置定时任务
```bash
cd /Users/candyhu/a-quant-system
chmod +x scripts/setup_scheduler.sh
./scripts/setup_scheduler.sh
```

#### 2. 手动测试
```bash
# 生成今日信号
python3 scripts/daily_signals.py

# 查看生成的信号
cat data/signals/signal_$(date +%Y%m%d).txt
```

#### 3. 定时任务会自动执行
- 时间: 每天下午3点30分（周一到周五）
- 输出: `data/signals/signal_YYYYMMDD.txt`
- 日志: `logs/scheduler.log`

## 📱 飞书消息推送（可选）

### 飞书Webhook配置

#### 1. 创建飞书机器人
1. 在飞书群聊中添加自定义机器人
2. 选择"Webhook"类型
3. 复制Webhook地址

#### 2. 配置推送
在 `scripts/daily_signals.py` 的 `main()` 函数中取消注释：
```python
# 替换为你的webhook地址
webhook_url = "https://open.feishu.cn/open-apis/bot/v2/hook/xxx"

# 发送消息
send_feishu_message(message, webhook_url)
```

#### 3. 测试推送
```bash
python3 scripts/daily_signals.py
```

## ⏰ 测试时间建议

### 短期测试 (1个月)
- 验证信号质量
- 检查推送稳定性
- 调整参数

### 中期测试 (3个月)
- 完整穿越不同市场环境
- 统计胜率、盈亏比
- 计算总收益

---

## 📈 测试期间监控指标

### 每周检查
- 信号数量（买入/卖出）
- ADX状态分布
- 持仓周转率

### 每月总结
- 总收益率
- 胜率
- 盈亏比
- 最大回撤

---

## 🎯 测试完成标准

### 通过标准
1. 总收益 > 基准（沪深300）
2. 胜率 > 50%
3. 盈亏比 > 1.0
4. 最大回撤 < 15%

### 失败信号
- 连续3个月亏损
- 最大回撤 > 20%
- 胜率 < 40%

---

## 🚨 注意事项

1. **交易成本**: 实盘有手续费、印花税、滑点
2. **涨跌停限制**: 可能无法按信号价格成交
3. **停牌风险**: 股票可能突然停牌无法交易
4. **数据延迟**: 模拟盘和实盘可能有时间差
