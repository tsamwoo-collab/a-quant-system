### 1. 持仓管理功能实现 ✅

**文件**: `scripts/position_manager.py`

创建持仓管理器，实现以下功能：
- 持仓数据持久化（JSON格式，`data/positions.json`）
- 添加持仓：记录入场价、股数、入场日期、信号强度
- 平仓：记录出场价、平仓日期、平仓原因
- 更新最高价：用于追踪止盈
- 检查止盈止损条件：
  - 初始止损：8%
  - 追踪止盈：8%（盈利3%后启用）
  - 目标止盈：30%（动态保护）

**核心类**:
```python
class PositionManager:

    def close_position(symbol, exit_price, reason)
    def check_stop_conditions(symbol, current_price, initial_stop, trailing_stop, take_profit)
    def get_positions_summary(prices) -> List[Dict]
```
